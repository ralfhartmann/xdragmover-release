Confidentialité

Hébergement du site

Ce site est hébergé sur GitHub Pages (GitHub, Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, États-Unis). Lors de votre visite sur ce site, GitHub peut collecter et traiter des données techniques telles que votre adresse IP et votre comportement de navigation. Je n'ai aucun contrôle sur ce traitement, et je ne collecte, ne traite ni ne stocke moi-même aucune donnée personnelle via ce site. Voir la déclaration de confidentialité de GitHub pour plus de détails.

Aucun cookie, aucun traçage

Ce site lui-même ne dépose aucun cookie et ne contient aucun script d'analyse, de publicité ou de traçage tiers qui lui soit propre.

L'application elle-même

XDragMover ne collecte, ne transmet ni ne stocke aucune donnée personnelle. Sa seule activité réseau est une vérification de mise à jour optionnelle, désactivée par défaut, qui interroge le dépôt de release GitHub public de ce projet — voir le README de l'application pour plus de détails.
