What is xdragmover?

xdragmover is a small, fast and reliable macOS utility inspired by the classic KDE-style window resizing and moving tool originally published at corz.org . It brings intuitive mouse-based window control to macOS without bloated frameworks or background daemons.

Features

  • Lightweight: Minimal resource usage, instant response.
  • GPLv3: Fully open-source and community-friendly.
  • Native macOS: No hacks, no kernel extensions. (Uses some undocumented APIs.)
  • Configurable: Adjust hotkeys and behavior to your workflow.
