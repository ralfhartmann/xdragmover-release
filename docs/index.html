xdagmover

A free, open-source window resizing & moving tool for macOS.

Download

About

xdagmover is inspired by the excellent KDE-style window resizing and moving tool for Windows, originally published at corz.org .

Features

  • Lightweight and fast
  • GPLv3 licensed
  • Native macOS support
  • Configurable hotkeys
