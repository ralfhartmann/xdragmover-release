Privacidad

Alojamiento del sitio web

Este sitio web está alojado en GitHub Pages (GitHub, Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, EE. UU.). Al visitar este sitio, GitHub puede recopilar y procesar datos técnicos como tu dirección IP y tu comportamiento de navegación. No tengo ningún control sobre este procesamiento, y yo mismo no recopilo, proceso ni almaceno ningún dato personal a través de este sitio web. Consulta la Declaración de privacidad de GitHub para más información.

Sin cookies, sin seguimiento

Este sitio web en sí no establece cookies ni incluye analítica, publicidad ni scripts de seguimiento de terceros propios.

La app en sí

XDragMover no recopila, transmite ni almacena ningún dato personal. Su única actividad de red es una comprobación de actualizaciones opcional, desactivada por defecto, que consulta el repositorio público de releases de GitHub de este proyecto — consulta el propio README de la app para más información.
