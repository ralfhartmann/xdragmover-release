Privacidad

Alojamiento de este sitio web

Este sitio web está alojado en GitHub Pages. El proveedor es GitHub, Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, EE. UU., o bien GitHub Ireland Limited, 70 Sir John Rogerson's Quay, Dublín 2, Irlanda.

Al visitar este sitio web se transmiten y procesan datos técnicos a GitHub, en particular su dirección IP, tipo de navegador, sistema operativo, URL de referencia, fecha y hora del acceso, así como los archivos solicitados. Estos datos se almacenan en registros del servidor y se utilizan para prestar, proteger y estabilizar el sitio web. Consulte la Declaración de privacidad de GitHub para más información.

DNS y Cloudflare

Los servidores de nombres de este dominio están completamente delegados a Cloudflare. Como parte de la resolución DNS, Cloudflare procesa datos personales, en particular direcciones IP y consultas DNS, técnicamente necesarias para resolver el dominio y hacer accesible el sitio web. El proveedor es Cloudflare, Inc., 101 Townsend St., San Francisco, CA 94107, EE. UU., así como Cloudflare Germany GmbH, Rosental 7, 80331 Múnich, Alemania.

La transferencia a EE. UU. se basa en el EU-US Data Privacy Framework y cláusulas contractuales tipo adicionales. Consulte la Política de privacidad de Cloudflare para más información.

Sin cookies, sin seguimiento

Este sitio web en sí no establece cookies ni incluye analítica, publicidad ni scripts de seguimiento de terceros propios. El procesamiento técnicamente necesario para entregar la página, como la resolución DNS o los registros del servidor, queda excluido y se describe arriba.

La app en sí

XDragMover no recopila, transmite ni almacena ningún dato personal. Su única actividad de red es una comprobación de actualizaciones opcional, desactivada por defecto, que consulta el repositorio público de releases de GitHub de este proyecto — consulte el propio README de la app para más información.

Apoyo a través de PayPal

El botón «Apoyar vía PayPal» de la página de inicio enlaza a paypal.me/ralf1234, una página gestionada por completo por PayPal, fuera de este sitio web. Yo no recibo ningún dato sobre esa visita — ni siquiera sé si hizo clic en el enlace. Todo lo que introduzca allí, incluido para realizar un pago, es procesado únicamente por PayPal según su propia política de privacidad; consulte la Declaración de privacidad de PayPal para más información.

Bases legales y periodo de conservación

El procesamiento de datos técnicos para prestar el sitio web se basa en el art. 6, apartado 1, letra f, del RGPD (interés legítimo: funcionamiento, seguridad y estabilidad del sitio web). Los datos que usted transmitiera activamente se procesarían con base en el art. 6, apartado 1, letra a, del RGPD (consentimiento) o la letra b (medidas precontractuales), según la finalidad.

Los datos de registro (logs) suelen conservarse solo brevemente. GitHub y Cloudflare pueden conservar determinados registros durante períodos variables; normalmente los registros operativos y de seguridad se eliminan en un plazo de horas a días. Los plazos exactos se rigen por las políticas de privacidad de cada proveedor.

Sus derechos

Tiene derecho de acceso, rectificación, supresión, limitación del tratamiento, oposición y portabilidad de los datos. Para ejercer sus derechos o por cualquier consulta sobre privacidad, póngase en contacto conmigo a través de la dirección de correo indicada en el Aviso legal.

Contacto y responsable

El responsable del tratamiento de datos en el sentido del RGPD es la persona o entidad mencionada en el Aviso legal. Consulte allí los datos de contacto.

Notas adicionales para visitantes internacionales

Las secciones anteriores ya describen de forma exhaustiva qué datos se procesan — únicamente los datos técnicos de acceso de GitHub Pages y Cloudflare, sin cookies, sin seguimiento y sin venta ni cesión de datos. Lo siguiente lo contextualiza para visitantes de jurisdicciones concretas.

Unión Europea (RGPD). Para los visitantes de la UE, el RGPD descrito arriba ya se aplica en su totalidad — la finalidad, la base legal, el periodo de conservación y sus derechos están cubiertos en las secciones «Bases legales y periodo de conservación» y «Sus derechos» anteriores.

California (CCPA/CPRA). Este sitio web no vende ni «comparte» (en el sentido de la CCPA/CPRA) ningún dato personal, ni elabora perfiles de usuario. Por ello, no es necesario un enlace «Do Not Sell or Share My Personal Information», ya que no hay nada que vender ni compartir. Los visitantes de California conservan, no obstante, el derecho de acceso, supresión y no discriminación; dirija sus solicitudes a la dirección de contacto indicada en el Aviso legal.

Brasil (LGPD). Dado que este sitio web no recopila ni procesa datos personales más allá de lo técnicamente necesario, no se ha designado un encargado de protección de datos (DPO) independiente conforme a la LGPD. El responsable en el sentido de la LGPD es la persona mencionada en el Aviso legal; los interesados en Brasil pueden ejercer sus derechos a través de la dirección de contacto allí indicada.

China (PIPL). Este sitio web no se dirige específicamente a visitantes en China ni exporta datos personales desde China. Si aun así se accede desde China, el procesamiento se limita a los datos técnicos de acceso descritos arriba, gestionados por GitHub Pages y Cloudflare; no es necesaria una notificación separada de transferencia transfronteriza conforme a la PIPL al no existir recopilación activa de datos.

India (DPDP). Este sitio web no recaba consentimientos sobre datos personales y, por tanto, no necesita un aviso multilingüe independiente conforme a la Digital Personal Data Protection Act de la India. Los datos técnicos de acceso descritos arriba son el único procesamiento que también ocurre en un acceso desde India.
